
  # Stock Simulator App

  This is a code bundle for Stock Simulator App. The original project is available at https://www.figma.com/design/N28lb3ZIUHgMxRxanSAt22/Stock-Simulator-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  ## In folder Server
  Run `npm init -y`
  Run `npm install express mongodb dotenv cors`
  Run `node index.js` to start the server

  ## In the root folder
  `python app.py`

  ** Important add the api key in discord to .env (create this file in /server if its not there)
